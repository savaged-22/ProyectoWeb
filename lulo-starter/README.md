# Lulo Starter (Angular + Spring Boot)

ZIP con un esqueleto **organizado por features** para repartir HU sin pisarse.

- Backend: `backend/` (Spring Boot, Java 21)
- Frontend: `frontend/` (Angular standalone + routing)
- Docs: `docs/` (contrato API borrador)

> Nota: no incluye `node_modules` ni un proyecto Angular generado por CLI completo.
> Para arrancar, genera los proyectos (Angular CLI / Spring Initializr) y copia estas carpetas/archivos.
